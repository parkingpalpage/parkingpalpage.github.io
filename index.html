Features

In order to solve this problem, we are creating a mobile application. This application will have a host of features which place an emphasis on utility and user experience.

Universal compatability

To the extent which it is possible, the application will be universal among all popular platforms.

User security and privacy

The newest encryption protocols will be used to secure payments and withdrawls. In addition, user data will be kept on separate, effectively inacessible servers.

Feature customization

All versions of the application will have many customizable features for user convenience.

Browser support

In the near future, the service will be fully accessible through any web browser as a standalone website.

Strong incentives for trust

Users will be insured a certain amount for damages and user ratings will be extensively collected to identify malicious sellers.

Rewards for payment from profit

Users will be able to pay with credit from selling their own spaces at a bonus, rewarding consistent users.
